# Genesis-Gates

A professional post-login dashboard experience implemented with Next.js.

## Getting started

1. Install dependencies

   ```bash
   npm install
   ```

2. Run the development server

   ```bash
   npm run dev
   ```

3. Open the app

   Visit `http://localhost:3000` to access the login panel and dashboard.

## Notes
- Session state persists in `localStorage` for demo purposes.
- The dashboard includes KPI highlights, pipeline status, and a lightweight task list.
